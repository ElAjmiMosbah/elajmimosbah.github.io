# Portfolio Web Designer

Un template de portfolio moderne et responsive pour web designers et développeurs.

## 📁 Structure du Projet

```
Portfolio/
├── index.html       # Page d'accueil principale
├── style.css        # Feuille de style responsive
├── script.js        # JavaScript pour l'interactivité
└── README.md        # Ce fichier
```

## ✨ Fonctionnalités

- **Design Moderne** : Interface épurée et professionnelle
- **Responsive** : Adapté à tous les appareils (mobile, tablette, desktop)
- **Sections Complètes** :
  - Accueil avec call-to-action
  - Compétences avec barres de progression
  - Services proposés
  - Portfolio de projets
  - Formulaire de contact
  - Liens sociaux

## 🎨 Personnalisation

### Couleurs
Modifiez les variables CSS dans `style.css` :
```css
:root {
    --primary-color: #667eea;      /* Couleur principale */
    --secondary-color: #764ba2;    /* Couleur secondaire */
    --text-dark: #1a202c;          /* Texte sombre */
    --text-light: #718096;         /* Texte clair */
    --bg-light: #f7fafc;           /* Fond clair */
}
```

### Contenu
1. Remplacez le texte par défaut par vos informations
2. Mettez à jour les compétences et leurs pourcentages
3. Ajoutez vos projets dans la section "Mes Projets"
4. Configurez vos liens sociaux dans la section contact

## 🚀 Comment Utiliser

1. **Ouvrir le fichier** : Double-cliquez sur `index.html` dans votre navigateur
2. **Modifier le contenu** : Éditez les fichiers avec un éditeur de texte
3. **Tester** : Rafraîchissez la page pour voir les changements

## 📝 Section par Section

### Hero Section
Modifiez le titre et la description pour une première impression personnalisée.

### Compétences
Ajoutez ou modifiez les compétences avec leurs pourcentages dans la section `skills-grid`.

### Services
Décrivez les services que vous proposez avec des icônes appropriées.

### Projets
Remplacez les projets d'exemple par vos réalisations réelles avec des images.

### Contact
Le formulaire affiche actuellement un message d'alerte. Pour un vrai contact, intégrez:
- Un service email (Formspree, EmailJS)
- Un backend Node.js
- Un service cloud (Firebase, Supabase)

## 🔗 Intégrations Recommandées

- **Icônes** : Font Awesome (déjà inclus via CDN)
- **Email** : [Formspree](https://formspree.io/), [EmailJS](https://www.emailjs.com/)
- **Hébergement** : Netlify, Vercel, GitHub Pages

## 📱 Responsive Design

Le site s'adapte automatiquement à :
- Desktop (1200px+)
- Tablette (768px - 1199px)
- Mobile (<768px)
